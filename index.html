What is JustType?

JustType is a sass mixin that automates good typography. Feed it base values, such as font-size, and it'll generate typographic styles that scale beautifully to any display. This demo uses a stylesheet created with the default values.

Who's it for?

It's for all you hacker-bloggers, itching to publish thoughts to the web (perhaps using jekyll and github pages?). Let JustType automate the typography, so you can focus on what matters: writing the content.

How about markdown?

JustType covers all html that can be generated from markdown. As a bonus, the default font stack is web-safe:

  1. Serif
    • Georgia
  2. Sans-serif
    • Trebuchet MS
  3. Monospace
    • Andale Mono
    • Lucida Console

Where's h5 and h6?

If a blog post needs an h5, I bet it also needs revising. If you desperately need an h6 though, it's easy to extend the style. Add to lib/typography-*.scss like so:

h5,
h6 {
    // your
    // style
    // here
}
